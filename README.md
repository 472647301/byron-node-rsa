# react-native-byron-rsa
适用于react-native的node-rsa模块

### 安装
 
```
npm i react-native-byron-rsa or yarn add react-native-byron-rsa
```

### 本地运行

1.clone 
```
git clone https://github.com/472647301/react-native-byron-rsa.git
```
2.安装依赖
```
cd react-native-byron-rsa/
npm i
```
3.测试
```
node test.js
```
